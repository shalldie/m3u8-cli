# m3u8-cli

Cli to download and compile m3u8 files.

## Installation

```bash
npm i m3u8-cli -g
```

## Usage

```bash
& m3u8-cli -h

选项：
  -u, --url      下载地址                                                 [必需]
  -o, --output   保存到的文件                                             [必需]
  -h, --help     显示帮助信息                                             [布尔]
  -v, --version  显示版本号                                               [布尔]

示例：
  m3u8-cli download -u [url] -o [output]
```
